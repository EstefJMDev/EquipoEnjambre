# AGENTS Mirror

La versión normativa canónica está en [../AGENTS.md](../AGENTS.md).

Este archivo se mantiene por compatibilidad para quien navegue la carpeta
`/agents`. Antes de trabajar en el repositorio debes leer `../AGENTS.md` y
después los mandatos específicos de cada agente en esta carpeta.

## Regla de uso

* `../AGENTS.md` gobierna el comportamiento global del repositorio.
* Los archivos de esta carpeta gobiernan el mandato de cada agente.
* Si este archivo y `../AGENTS.md` divergen, prevalece `../AGENTS.md`.
