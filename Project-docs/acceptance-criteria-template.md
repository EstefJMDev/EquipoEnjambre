# FlowWeaver - Acceptance Criteria Template

## Uso

Plantilla para definir criterios de aceptación verificables sobre entregables del
sistema multiagente.

## Reglas

* cada criterio debe ser verificable
* cada criterio debe pertenecer a una fase concreta
* deben listarse exclusiones explícitas
* debe existir método de validación

## Plantilla

```md
# Acceptance Criteria Set

item:
phase:
owner_agent:

## Validates

## Does Not Validate

## Criteria

- [ ] criterio 1
- [ ] criterio 2
- [ ] criterio 3

## Exclusions

## Validation Method

## Blocking Risks
```
