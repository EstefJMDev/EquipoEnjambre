# FlowWeaver — Visión

FlowWeaver es un asistente digital que entiende el contexto de trabajo del usuario y prepara el siguiente paso antes de que este tenga que pedirlo.

La promesa del producto no es automatizar tareas mediante reglas manuales, sino detectar episodios reales de intención y preparar un workspace útil con mínima fricción.

La forma más clara de esa promesa en el MVP es el puente móvil → desktop:

* el usuario guarda varios recursos relacionados desde el móvil
* FlowWeaver detecta el episodio
* el escritorio recibe la señal
* el workspace aparece ya preparado

La visión a largo plazo es que el sistema pase de detectar episodios puntuales a reconocer hábitos recurrentes y actuar con confianza progresiva, sin sacrificar privacidad ni control del usuario.

## Principios rectores

* anticipación antes que reacción
* cero reglas manuales
* privacidad verificable por diseño
* valor inmediato desde el primer uso
* confianza progresiva, no automatización agresiva
