# FlowWeaver - Task Template

## Uso

Plantilla para tareas documentales del sistema multiagente. No usar para
tickets de implementación funcional del producto.

## Campos obligatorios

* `task_id`
* `title`
* `phase`
* `owner_agent`
* `objective`
* `documents_to_read`
* `in_scope`
* `out_of_scope`
* `acceptance_criteria`
* `risks`
* `required_handoff`

## Plantilla

```md
# Task

task_id:
title:
phase:
owner_agent:

## Objective

## Documents To Read

## In Scope

## Out Of Scope

## Acceptance Criteria

## Risks

## Required Handoff
```
