# FlowWeaver — Deliverable Map

Mapea qué tipos de entregables puede producir cada agente y en qué fase.

Tipos de entregable permitidos en este repo:
- documento normativo
- propuesta de cambio
- auditoría
- matriz
- plantilla
- handoff
- checklist
- revisión de coherencia
- análisis de riesgo
- cierre de fase

Tipos de entregable prohibidos en este repo:
- código funcional del producto
- módulos del producto
- apps del producto
- servicios del producto
- infraestructura del producto

Objetivo:
Asegurar que el enjambre construye gobernanza, no implementación.
